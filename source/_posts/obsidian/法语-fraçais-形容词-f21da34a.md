## 1. 形容词要配合名词的阴阳性、单复数

法语形容词通常要和它修饰的名词保持一致：

|名词|形容词|
|---|---|
|阳性单数|petit|
|阴性单数|petite|
|阳性复数|petits|
|阴性复数|petites|

比如：

```
un petit appartement一个小公寓une petite maison一栋小房子des petits appartements一些小公寓des petites maisons一些小房子
```

最基本规律是：

```
阳性 → 阴性：通常加 -e单数 → 复数：通常加 -s
```

但有很多特殊变化。

---

## 2. 阴性变化：通常加 -e，但有特殊形式

### 普通变化

```
grand → grandepetit → petitejoli → jolie
```

例子：

```
un grand salon一个大客厅une grande chambre一个大卧室
```

### 以 -e 结尾的形容词，阴阳性一样

```
rouge → rougejeune → jeunefacile → facile
```

比如：

```
un livre rouge一本红色的书une voiture rouge一辆红色的车
```

### 常见特殊变化

|阳性|阴性|意思|
|---|---|---|
|bon|bonne|好的|
|gros|grosse|胖的/大的|
|blanc|blanche|白色的|
|long|longue|长的|
|vieux|vieille|老的/旧的|
|beau|belle|美的|
|nouveau|nouvelle|新的|

比如：

```
un bon café一杯好咖啡une bonne idée一个好主意un vieux livre一本旧书une vieille maison一栋老房子
```

---

## 3. 复数变化：通常加 -s

```
petit → petitsgrande → grandesjoli → jolisbelle → belles
```

比如：

```
des chambres grandes一些大的房间des voitures rouges一些红色的车
```

但是有些形容词本来就以 `-s` 或 `-x` 结尾，复数不变：

```
un homme heureux一个快乐的男人des hommes heureux一些快乐的男人
```

---

## 4. 形容词的位置：大多数放在名词后面

法语形容词和英语不一样，**大多数形容词放在名词后面**。

```
une voiture rouge一辆红色的车un appartement moderne一个现代化的公寓une chambre confortable一个舒适的房间
```

尤其是这些一般放后面：

颜色：

```
une robe bleue一条蓝裙子
```

国籍：

```
un étudiant chinois一个中国学生
```

形状：

```
une table ronde一张圆桌
```

宗教、风格、材料、性质也常放后面。

---

## 5. 有些常用短形容词放在名词前面

有一批很常见、很基础的形容词，一般放在名词前面。

可以记成：**美、年龄、好坏、大小、数量**这一类。

|形容词|意思|
|---|---|
|beau / belle|美的|
|joli / jolie|漂亮的|
|jeune|年轻的|
|vieux / vieille|老的/旧的|
|bon / bonne|好的|
|mauvais / mauvaise|坏的|
|grand / grande|大的/高的|
|petit / petite|小的|
|nouveau / nouvelle|新的|
|premier / première|第一的|

比如：

```
une belle maison一栋漂亮的房子un petit appartement一个小公寓un bon restaurant一家好餐厅une nouvelle voiture一辆新车
```

这个地方很重要：不是所有形容词都后置，有些高频词就是前置。

---

## 6. 有些形容词前置和后置意思不同

这部分很法语，很阴间，但也很有味儿 🫠

比如：

```
un grand homme一个伟大的人un homme grand一个高个子男人
```

```
un ancien professeur以前的老师un professeur ancien年纪很老的老师 / 古老的老师
```

```
un pauvre homme一个可怜的人un homme pauvre一个贫穷的人
```

```
ma propre chambre我自己的房间une chambre propre一个干净的房间
```

所以位置不只是语法问题，有时候会改变意思。

---

## 7. 形容词也可以放在 être 后面作表语

类似英语里的：

```
The room is big.
```

法语是：

```
La chambre est grande.这个房间很大。
```

注意，哪怕形容词不直接挨着名词，也还是要配合名词：

```
Le salon est grand.客厅很大。La maison est grande.房子很大。Les chambres sont grandes.这些房间很大。
```

---

## 8. 常见句型：C’est + 形容词

这个很常见：

```
C’est cher.这很贵。C’est beau.这很美。C’est difficile.这很难。C’est intéressant.这很有意思。
```

如果后面是具体名词，就要注意：

```
C’est un bon restaurant.这是一家好餐厅。C’est une belle maison.这是一栋漂亮的房子。
```

---

## 9. 形容词的比较级

### 更……

```
plus + 形容词 + que
```

比如：

```
Paris est plus grand que Lyon.巴黎比里昂大。
```

### 不如……

```
moins + 形容词 + que
```

```
Cette chambre est moins chère que cette maison.这个房间比这栋房子便宜。
```

### 和……一样

```
aussi + 形容词 + que
```

```
Il est aussi intelligent que son frère.他和他哥哥一样聪明。
```

---

## 10. 最高级

### 最……

```
le / la / les plus + 形容词
```

比如：

```
C’est le plus beau jardin.这是最漂亮的花园。
```

```
C’est la plus grande chambre.这是最大的房间。
```

### 最不……

```
le / la / les moins + 形容词
```

```
C’est le moins cher.这是最便宜的。
```

注意 `le / la / les` 要看后面的名词：

```
le plus grand appartement最大的公寓la plus grande maison最大的房子les plus grandes chambres最大的房间们
```

---

## 11. 一些形容词有特殊比较级

最重要的是：

```
bon → meilleur
```

不是 `plus bon`。

```
Ce restaurant est meilleur.这家餐厅更好。
```

```
C’est le meilleur restaurant.这是最好的餐厅。
```

阴阳单复数也要变：

|形式|用法|
|---|---|
|meilleur|阳性单数|
|meilleure|阴性单数|
|meilleurs|阳性复数|
|meilleures|阴性复数|

比如：

```
une meilleure idée一个更好的主意les meilleures chambres最好的房间
```

---

## 12. 所有格形容词：mon, ma, mes 这一类

这个也叫**物主形容词**，表示“我的、你的、他的”。

|中文|阳性单数|阴性单数|复数|
|---|---|---|---|
|我的|mon|ma|mes|
|你的|ton|ta|tes|
|他/她/它的|son|sa|ses|
|我们的|notre|notre|nos|
|你们的/您的|votre|votre|vos|
|他们/她们的|leur|leur|leurs|

比如：

```
mon livre我的书ma chambre我的房间mes amis我的朋友们son appartement他/她的公寓ses parents他/她的父母
```

注意：看的是**后面名词的性数**，不是拥有者的性别。

```
son père他/她的爸爸sa mère他/她的妈妈
```

---

## 13. 指示形容词：ce, cette, ces

表示“这个、那个、这些、那些”。

|形式|用法|
|---|---|
|ce|阳性单数|
|cet|阳性单数，后接元音或哑音 h|
|cette|阴性单数|
|ces|复数|

比如：

```
ce livre这本书cet appartement这套公寓cette maison这栋房子ces chambres这些房间
```

---

## 14. 不定形容词：quelques, plusieurs, tout 等

这些也是形容词性质的词，会修饰名词。

```
quelques amis几个朋友plusieurs livres好几本书tous les jours每天toute la maison整个房子
```

`tout` 变化比较多：

|形式|用法|
|---|---|
|tout|阳性单数|
|toute|阴性单数|
|tous|阳性复数|
|toutes|阴性复数|

比如：

```
tout le livre整本书toute la journée一整天tous les étudiants所有学生toutes les chambres所有房间
```

---

## 15. 疑问形容词：quel

表示“哪个、什么样的”。

|形式|用法|
|---|---|
|quel|阳性单数|
|quelle|阴性单数|
|quels|阳性复数|
|quelles|阴性复数|

比如：

```
Quel âge avez-vous ?您多大？Quelle est votre adresse ?您的地址是什么？Quels livres aimez-vous ?你喜欢哪些书？Quelles langues parlez-vous ?你说哪些语言？
```
