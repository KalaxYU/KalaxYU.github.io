# Un message après le cours
Blog permalink: [https://kalax524.com/lectures-francaises/2026-07-18-message-apres-cours/](https://kalax524.com/lectures-francaises/2026-07-18-message-apres-cours/)
## Niveau

A1-A2. Texte original court pour lecture quotidienne.

## Texte

Après le cours de français, Mei reste quelques minutes dans la salle. Elle veut poser une question, mais le professeur parle déjà avec un autre étudiant. Mei ouvre son téléphone et écrit un message très simple. Elle dit bonjour, explique la question et remercie le professeur. La question porte sur la différence entre “à” et “de”. Puis elle relit le message pour vérifier les accents et les verbes. Elle change deux mots, parce qu’elle veut être claire. Elle ajoute aussi une phrase courte: “Pouvez-vous donner un exemple?” Le soir, le professeur répond avec trois exemples. Les exemples sont simples, mais ils montrent bien la règle. Mei copie les exemples dans son cahier et fait une petite liste. Elle écrit aussi une phrase personnelle avec chaque exemple. Le lendemain, elle relit la liste avant le petit déjeuner. Elle comprend que bien poser une question est déjà une partie de l’apprentissage.

## Vocabulaire

## Grammaire

| Point | Explication |
| --- | --- |
| vouloir + infinitif | veut poser = 想要提问。 |
| avec + nom | avec un autre étudiant = 和另一个学生。 |
| pour + infinitif | pour vérifier = 为了检查。 |
| que 引导宾语从句 | Elle comprend que... = 她明白…… |
