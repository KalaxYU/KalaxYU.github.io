# 复分析手写笔记整理

> 说明：本版是在 OCR 原稿基础上做的**逻辑校订版**。原始逐页 OCR 已备份为 [复分析手写笔记整理_OCR原稿](/notes/数学-复分析手写笔记整理_ocr原稿-b12a93c3/)。以下内容不再逐页保留 OCR 错字，而是按数学主线重排、修正常见错误、补全必要证明步骤，并加入初学者提示。

## 0. 这份笔记在讲什么

这份笔记大致有两条主线：

1. **复分析主线**：复数与拓扑 -> 全纯函数 -> Cauchy 积分定理与公式 -> Laurent 展开与留数 -> 共形映射。
2. **多值函数主线**：分支点与分支割线 -> 单值分支 -> 解析延拓与单值化 -> Riemann 曲面与覆盖。

> [!tip] 初学者怎么看
> 先把复分析主线学顺，再看 Riemann 曲面。Riemann 曲面本质上是为了解决“<span class="kalax-inline-math">\sqrt z</span>、<span class="kalax-inline-math">\log z</span>、<span class="kalax-inline-math">z^a</span> 这些多值函数到底是不是函数”的问题：把多个值拆到不同层上，让它们在新空间里变成单值全纯函数。

### 本次校订修正的主要问题

- 将 <span class="kalax-inline-math">\mathbb C</span> 写成 <span class="kalax-inline-math">\mathbb R[x]/(x^2+1)</span>，而不是 OCR 中的 <span class="kalax-inline-math">\mathbb P[x]</span> 或 <span class="kalax-inline-math">\mathbb Z[z]</span>。
- 修正 Cauchy 列定义：应为 <span class="kalax-inline-math">|z_n-z_m|\to 0</span>，不是 <span class="kalax-inline-math">|z_n-w|\to 0</span>。
- 修正 Cauchy-Riemann 方程：若 <span class="kalax-inline-math">f=u+iv</span>，则

<div class="kalax-display-math">u_x=v_y,\qquad u_y=-v_x.</div>

- 修正 Cauchy 积分公式：

<div class="kalax-display-math">f(a)=\frac1{2\pi i}\int_C\frac{f(z)}{z-a}\,dz,</div>

不是把分母变量和被求点写混。

- 修正高阶导数公式：

<div class="kalax-display-math">f^{(n)}(a)=\frac{n!}{2\pi i}\int_C\frac{f(z)}{(z-a)^{n+1}}\,dz.</div>

- 将“branch point”“pole”“singularity”区分清楚：例如 <span class="kalax-inline-math">z^{-1}</span> 在 <span class="kalax-inline-math">0</span> 是极点，不是分支点；<span class="kalax-inline-math">\sqrt z</span> 在 <span class="kalax-inline-math">0,\infty</span> 是分支点。
- 将 Riemann 曲面的定义改成“带复结构的一维复流形”，并补足 atlas、transition map、holomorphic map 的定义。

## 1. 复数、序与基本拓扑

### 复数域的构造

复数可以由实系数多项式环构造：

<div class="kalax-display-math">\mathbb C\cong \mathbb R[x]/(x^2+1).</div>

在商环中，记 <span class="kalax-inline-math">i=[x]</span>，则 <span class="kalax-inline-math">i^2=-1</span>。于是每个复数都可唯一写为

<div class="kalax-display-math">z=x+iy,\qquad x,y\in\mathbb R.</div>

> [!note] 这在讲什么
> 这说明复数并不是凭空加出来的对象，而是把实数域扩张，使方程 <span class="kalax-inline-math">x^2+1=0</span> 有解。这个“给多项式补根”的思想后来会在代数和 Riemann 曲面里反复出现。

### 基本运算与几何意义

若

<div class="kalax-display-math">z_1=x_1+iy_1,\qquad z_2=x_2+iy_2,</div>

则

<div class="kalax-display-math">z_1+z_2=(x_1+x_2)+i(y_1+y_2),</div>

<div class="kalax-display-math">z_1z_2=(x_1x_2-y_1y_2)+i(x_1y_2+x_2y_1).</div>

共轭与模长定义为

<div class="kalax-display-math">\bar z=x-iy,\qquad |z|=\sqrt{x^2+y^2}.</div>

并且

<div class="kalax-display-math">z\bar z=|z|^2.</div>

几何上，<span class="kalax-inline-math">z=x+iy</span> 是平面上的点 <span class="kalax-inline-math">(x,y)</span>，也可看作从原点指向该点的向量。若 <span class="kalax-inline-math">z\ne0</span>，可以写成极坐标形式

<div class="kalax-display-math">z=re^{i\theta},\qquad r=|z|,\quad \theta=\arg z.</div>

Euler 公式为

<div class="kalax-display-math">e^{i\theta}=\cos\theta+i\sin\theta.</div>

由此得到 De Moivre 公式：

<div class="kalax-display-math">(re^{i\theta})^n=r^n e^{in\theta}.</div>

特别地，<span class="kalax-inline-math">n</span> 次单位根为

<div class="kalax-display-math">z_k=e^{2\pi i k/n},\qquad k=0,1,\dots,n-1.</div>

它们均匀分布在单位圆上。

### <span class="kalax-inline-math">\mathbb C</span> 不能成为有序域

**命题。** 不存在一个全序 <span class="kalax-inline-math">\le</span> 使 <span class="kalax-inline-math">\mathbb C</span> 成为有序域。

**证明。** 在任意有序域中，任意元素平方非负。因为若 <span class="kalax-inline-math">a\ge0</span>，则 <span class="kalax-inline-math">a^2\ge0</span>；若 <span class="kalax-inline-math">a&lt;0</span>，则 <span class="kalax-inline-math">-a&gt;0</span>，所以 <span class="kalax-inline-math">a^2=(-a)^2\ge0</span>。特别地，<span class="kalax-inline-math">1=1^2&gt;0</span>，并且 <span class="kalax-inline-math">i^2\ge0</span>。但 <span class="kalax-inline-math">i^2=-1</span>，于是 <span class="kalax-inline-math">-1\ge0</span>。两边加 <span class="kalax-inline-math">1</span> 得 <span class="kalax-inline-math">0\ge1</span>，这与 <span class="kalax-inline-math">1&gt;0</span> 矛盾。因此 <span class="kalax-inline-math">\mathbb C</span> 不是有序域。

> [!tip] 初学者提示
> 实数里有“正负大小”，复数平面里没有与乘法兼容的全局大小。复分析里更多使用的是模长 <span class="kalax-inline-math">|z|</span>，而不是 <span class="kalax-inline-math">z_1&lt;z_2</span>。

### 序列、开集、闭集与紧性

复数列 <span class="kalax-inline-math">z_n</span> 收敛到 <span class="kalax-inline-math">w</span>，记作 <span class="kalax-inline-math">z_n\to w</span>，若

<div class="kalax-display-math">\lim_{n\to\infty}|z_n-w|=0.</div>

复数列是 Cauchy 列，若对任意 <span class="kalax-inline-math">\varepsilon&gt;0</span>，存在 <span class="kalax-inline-math">N</span>，使得当 <span class="kalax-inline-math">m,n&gt;N</span> 时

<div class="kalax-display-math">|z_n-z_m|&lt;\varepsilon.</div>

由于 <span class="kalax-inline-math">\mathbb C\cong\mathbb R^2</span> 是完备的，复数列收敛当且仅当它是 Cauchy 列。

开圆盘与闭圆盘记作

<div class="kalax-display-math">D_r(z_0)=\{z\in\mathbb C:|z-z_0|&lt;r\},</div>

<div class="kalax-display-math">\overline D_r(z_0)=\{z\in\mathbb C:|z-z_0|\le r\}.</div>

集合 <span class="kalax-inline-math">U\subset\mathbb C</span> 是开集，若每个点 <span class="kalax-inline-math">z\in U</span> 都有一个小圆盘 <span class="kalax-inline-math">D_r(z)\subset U</span>。集合 <span class="kalax-inline-math">F</span> 是闭集，若其补集 <span class="kalax-inline-math">\mathbb C\setminus F</span> 是开集。

在 <span class="kalax-inline-math">\mathbb C</span> 中，集合紧当且仅当它闭且有界。这是 Heine-Borel 定理在 <span class="kalax-inline-math">\mathbb R^2</span> 中的形式。

### 连通与道路连通

集合 <span class="kalax-inline-math">U</span> 连通，表示不能写成两个非空不交开集的并。集合 <span class="kalax-inline-math">U</span> 道路连通，若任意两点 <span class="kalax-inline-math">z_0,z_1\in U</span> 都可由连续路径

<div class="kalax-display-math">\gamma:[0,1]\to U,\qquad \gamma(0)=z_0,\quad \gamma(1)=z_1</div>

连接。

> [!note] 为什么要学拓扑
> Cauchy 定理、单值分支、解析延拓都依赖区域是否连通、单连通、有无洞。复分析的很多现象不是局部微分决定的，而是由区域的全局拓扑决定的。

## 2. 全纯函数与 Cauchy-Riemann 方程

### 全纯与复可微

设 <span class="kalax-inline-math">U\subset\mathbb C</span> 为开集，函数 <span class="kalax-inline-math">f:U\to\mathbb C</span> 在 <span class="kalax-inline-math">z_0\in U</span> 复可微，若极限

<div class="kalax-display-math">f'(z_0)=\lim_{h\to0}\frac{f(z_0+h)-f(z_0)}{h}</div>

存在。若 <span class="kalax-inline-math">f</span> 在 <span class="kalax-inline-math">U</span> 中每一点复可微，则称 <span class="kalax-inline-math">f</span> 在 <span class="kalax-inline-math">U</span> 上全纯，或 holomorphic。

> [!warning] 易错点
> 复可微比实二维可微强得多。实函数 <span class="kalax-inline-math">u(x,y),v(x,y)</span> 可微并不自动推出 <span class="kalax-inline-math">f=u+iv</span> 复可微；还必须满足 Cauchy-Riemann 方程。

### Cauchy-Riemann 方程

写

<div class="kalax-display-math">f(z)=f(x+iy)=u(x,y)+iv(x,y).</div>

若 <span class="kalax-inline-math">f</span> 在 <span class="kalax-inline-math">z_0</span> 复可微，且 <span class="kalax-inline-math">u,v</span> 在 <span class="kalax-inline-math">z_0</span> 附近可微，则有

<div class="kalax-display-math">u_x=v_y,\qquad u_y=-v_x.</div>

反过来，若 <span class="kalax-inline-math">u,v</span> 的一阶偏导数连续，并满足上式，则 <span class="kalax-inline-math">f</span> 全纯。

**证明思路。** 复导数要求从任何方向趋近都得到同一极限。沿实轴方向 <span class="kalax-inline-math">h\in\mathbb R</span>，

<div class="kalax-display-math">f'(z)=u_x+iv_x.</div>

沿虚轴方向 <span class="kalax-inline-math">h=it</span>，

<div class="kalax-display-math">f'(z)=\frac{u(x,y+t)-u(x,y)+i(v(x,y+t)-v(x,y))}{it}
=v_y-iu_y.</div>

两者相等，得到

<div class="kalax-display-math">u_x=v_y,\qquad v_x=-u_y.</div>

即 Cauchy-Riemann 方程。

### 调和函数

若实值函数 <span class="kalax-inline-math">u(x,y)</span> 满足

<div class="kalax-display-math">\Delta u=u_{xx}+u_{yy}=0,</div>

则称 <span class="kalax-inline-math">u</span> 为调和函数。

若 <span class="kalax-inline-math">f=u+iv</span> 全纯，且二阶偏导连续，则 <span class="kalax-inline-math">u</span> 与 <span class="kalax-inline-math">v</span> 都调和。

**证明。** 由 Cauchy-Riemann 方程 <span class="kalax-inline-math">u_x=v_y</span>，<span class="kalax-inline-math">u_y=-v_x</span>。两边求导：

<div class="kalax-display-math">u_{xx}=v_{yx},\qquad u_{yy}=-v_{xy}.</div>

若二阶混合偏导相等，则

<div class="kalax-display-math">u_{xx}+u_{yy}=v_{yx}-v_{xy}=0.</div>

同理 <span class="kalax-inline-math">v_{xx}+v_{yy}=0</span>。

> [!tip] 初学者提示
> 全纯函数同时包含两个调和函数：实部和虚部。这就是复分析和二维势理论、流体、静电之间的桥梁。

### 幂级数

幂级数

<div class="kalax-display-math">\sum_{n=0}^{\infty}a_n(z-z_0)^n</div>

存在收敛半径 <span class="kalax-inline-math">R\in[0,\infty]</span>，满足：

- 当 <span class="kalax-inline-math">|z-z_0|&lt;R</span> 时绝对收敛；
- 当 <span class="kalax-inline-math">|z-z_0|&gt;R</span> 时发散；
- 在 <span class="kalax-inline-math">|z-z_0|=R</span> 上需要单独判断。

收敛半径可由 Cauchy-Hadamard 公式给出：

<div class="kalax-display-math">\frac1R=\limsup_{n\to\infty}|a_n|^{1/n}.</div>

在收敛圆盘内，幂级数可以逐项求导、逐项积分，并定义一个全纯函数。

> [!note] 关键逻辑
> 后面 Taylor 展开、Laurent 展开、留数定理都依赖这一点：在复分析里，全纯函数天然可以展开成幂级数。

## 3. 路径积分、Cauchy 定理与 Cauchy 公式

### 路径积分

设 <span class="kalax-inline-math">\gamma:[a,b]\to U</span> 是分段 <span class="kalax-inline-math">C^1</span> 路径，<span class="kalax-inline-math">f</span> 连续。定义

<div class="kalax-display-math">\int_\gamma f(z)\,dz
=\int_a^b f(\gamma(t))\gamma'(t)\,dt.</div>

若 <span class="kalax-inline-math">\gamma</span> 是闭路径，则 <span class="kalax-inline-math">\gamma(a)=\gamma(b)</span>。

### 原函数与积分为零

若 <span class="kalax-inline-math">f</span> 在 <span class="kalax-inline-math">U</span> 上有原函数 <span class="kalax-inline-math">F</span>，即 <span class="kalax-inline-math">F'=f</span>，则对任意路径 <span class="kalax-inline-math">\gamma</span>，

<div class="kalax-display-math">\int_\gamma f(z)\,dz=F(\gamma(b))-F(\gamma(a)).</div>

因此对任意闭路径 <span class="kalax-inline-math">\gamma</span>，

<div class="kalax-display-math">\int_\gamma f(z)\,dz=0.</div>

> [!tip] 初学者提示
> 这和微积分基本定理完全类似。Cauchy 理论的核心之一就是：在合适区域内，全纯函数的路径积分也像保守场一样只与端点有关。

### Goursat 定理与 Cauchy 积分定理

**Goursat 定理。** 若 <span class="kalax-inline-math">f</span> 在三角形及其内部全纯，则沿三角形边界的积分为零。

**Cauchy 积分定理。** 若 <span class="kalax-inline-math">U</span> 单连通，<span class="kalax-inline-math">f</span> 在 <span class="kalax-inline-math">U</span> 上全纯，则任意闭路径 <span class="kalax-inline-math">\gamma\subset U</span> 满足

<div class="kalax-display-math">\int_\gamma f(z)\,dz=0.</div>

更一般地，若两条闭路径在 <span class="kalax-inline-math">U</span> 内同伦，且 <span class="kalax-inline-math">f</span> 在 <span class="kalax-inline-math">U</span> 全纯，则它们的积分相等。

> [!note] 单连通在干什么
> 单连通表示区域没有洞。<span class="kalax-inline-math">\mathbb C\setminus\{0\}</span> 有洞，所以 <span class="kalax-inline-math">1/z</span> 在那里全纯，但
> <div class="kalax-display-math">&gt; \int_{|z|=1}\frac1z\,dz=2\pi i\ne0.
&gt;</div>
> 洞会贡献“绕数”，这正是留数和单值分支的来源。

### Cauchy 积分公式

设 <span class="kalax-inline-math">f</span> 在包含闭圆盘 <span class="kalax-inline-math">\overline D_R(a)</span> 的开集上全纯，<span class="kalax-inline-math">C=\partial D_R(a)</span> 取正向，则对圆盘内部任意 <span class="kalax-inline-math">w</span>，

<div class="kalax-display-math">f(w)=\frac1{2\pi i}\int_C\frac{f(z)}{z-w}\,dz.</div>

**证明要点。** 写成

<div class="kalax-display-math">\int_C\frac{f(z)}{z-w}\,dz
=\int_C\frac{f(z)-f(w)}{z-w}\,dz
+f(w)\int_C\frac1{z-w}\,dz.</div>

第一项中

<div class="kalax-display-math">\frac{f(z)-f(w)}{z-w}</div>

在 <span class="kalax-inline-math">w</span> 处可补成全纯函数，因此积分为零；第二项为

<div class="kalax-display-math">f(w)\cdot 2\pi i.</div>

所以得到公式。

### 高阶导数公式

同样条件下，<span class="kalax-inline-math">f</span> 任意阶可导，且

<div class="kalax-display-math">f^{(n)}(w)=\frac{n!}{2\pi i}\int_C\frac{f(z)}{(z-w)^{n+1}}\,dz.</div>

> [!tip] 初学者提示
> 这是复分析最神奇的地方之一：只要一次复可微，就自动无限次可微，甚至能由边界值决定内部所有导数。

### Cauchy 估计与 Taylor 展开

由高阶 Cauchy 公式可以得到 Cauchy 估计。若 <span class="kalax-inline-math">f</span> 在闭圆盘 <span class="kalax-inline-math">\overline D_R(a)</span> 的邻域全纯，且

<div class="kalax-display-math">M_R=\max_{|z-a|=R}|f(z)|,</div>

则

<div class="kalax-display-math">|f^{(n)}(a)|\le \frac{n!M_R}{R^n}.</div>

特别地，全纯函数在每个点附近都有 Taylor 展开：

<div class="kalax-display-math">f(z)=\sum_{n=0}^{\infty}\frac{f^{(n)}(a)}{n!}(z-a)^n.</div>

> [!note] 逻辑作用
> Cauchy 公式给导数，导数估计给 Taylor 展开。后面 Liouville 定理、正规族、Riemann 映射定理都会用到这种“边界控制内部”的思想。

### Liouville 定理与代数学基本定理

**Liouville 定理。** 若 <span class="kalax-inline-math">f</span> 在整个 <span class="kalax-inline-math">\mathbb C</span> 上全纯且有界，则 <span class="kalax-inline-math">f</span> 为常数。

**证明。** Cauchy 估计给出

<div class="kalax-display-math">|f'(a)|\le \frac{M_R}{R},</div>

其中 <span class="kalax-inline-math">M_R=\max_{|z-a|=R}|f(z)|</span>。若 <span class="kalax-inline-math">f</span> 有界，<span class="kalax-inline-math">M_R\le M</span>，令 <span class="kalax-inline-math">R\to\infty</span>，得 <span class="kalax-inline-math">f'(a)=0</span>。任意 <span class="kalax-inline-math">a</span> 都如此，所以 <span class="kalax-inline-math">f</span> 常数。

**代数学基本定理。** 每个非常数复系数多项式在 <span class="kalax-inline-math">\mathbb C</span> 中有根。

**证明。** 若多项式 <span class="kalax-inline-math">p</span> 无根，则 <span class="kalax-inline-math">1/p</span> 在 <span class="kalax-inline-math">\mathbb C</span> 上全纯。由于 <span class="kalax-inline-math">|p(z)|\to\infty</span> 当 <span class="kalax-inline-math">|z|\to\infty</span>，所以 <span class="kalax-inline-math">1/p</span> 有界。由 Liouville 定理，<span class="kalax-inline-math">1/p</span> 常数，矛盾。

## 4. Laurent 展开、孤立奇点与留数

### 孤立奇点

若 <span class="kalax-inline-math">f</span> 在穿孔邻域

<div class="kalax-display-math">0&lt;|z-a|&lt;R</div>

中全纯，则 <span class="kalax-inline-math">a</span> 是 <span class="kalax-inline-math">f</span> 的孤立奇点。孤立奇点分三类：

1. **可去奇点**：<span class="kalax-inline-math">f</span> 可补定义为全纯函数。
2. **极点**：存在 <span class="kalax-inline-math">m\ge1</span>，使 <span class="kalax-inline-math">(z-a)^m f(z)</span> 在 <span class="kalax-inline-math">a</span> 处可去且补值非零。
3. **本性奇点**：既不可去也不是极点。

### Laurent 展开

若 <span class="kalax-inline-math">f</span> 在环域

<div class="kalax-display-math">r&lt;|z-a|&lt;R</div>

中全纯，则

<div class="kalax-display-math">f(z)=\sum_{n=-\infty}^{\infty}c_n(z-a)^n.</div>

其中

<div class="kalax-display-math">c_n=\frac1{2\pi i}\int_C\frac{f(\zeta)}{(\zeta-a)^{n+1}}\,d\zeta.</div>

负幂部分

<div class="kalax-display-math">\sum_{n=1}^{\infty}c_{-n}(z-a)^{-n}</div>

称为 principal part。

### 留数

Laurent 展开中 <span class="kalax-inline-math">(z-a)^{-1}</span> 的系数称为留数：

<div class="kalax-display-math">\operatorname{Res}(f,a)=c_{-1}.</div>

等价地，

<div class="kalax-display-math">\operatorname{Res}(f,a)=\frac1{2\pi i}\int_C f(z)\,dz.</div>

若 <span class="kalax-inline-math">a</span> 是简单极点，则

<div class="kalax-display-math">\operatorname{Res}(f,a)=\lim_{z\to a}(z-a)f(z).</div>

若

<div class="kalax-display-math">f(z)=\frac{g(z)}{h(z)},\qquad h(a)=0,\quad h'(a)\ne0,</div>

则 <span class="kalax-inline-math">a</span> 是简单极点，且

<div class="kalax-display-math">\operatorname{Res}(f,a)=\frac{g(a)}{h'(a)}.</div>

### 留数定理

设 <span class="kalax-inline-math">f</span> 在区域 <span class="kalax-inline-math">U</span> 中除有限多个孤立奇点 <span class="kalax-inline-math">a_1,\dots,a_n</span> 外全纯，<span class="kalax-inline-math">\gamma</span> 是不经过这些奇点的正向闭曲线，则

<div class="kalax-display-math">\int_\gamma f(z)\,dz
=2\pi i\sum_{k=1}^n n(\gamma,a_k)\operatorname{Res}(f,a_k),</div>

其中 <span class="kalax-inline-math">n(\gamma,a_k)</span> 是 <span class="kalax-inline-math">\gamma</span> 绕 <span class="kalax-inline-math">a_k</span> 的绕数。

若 <span class="kalax-inline-math">\gamma</span> 简单正向包围所有 <span class="kalax-inline-math">a_k</span>，则

<div class="kalax-display-math">\int_\gamma f(z)\,dz
=2\pi i\sum_{k=1}^n\operatorname{Res}(f,a_k).</div>

> [!tip] 初学者提示
> 留数定理把复杂积分变成“找 <span class="kalax-inline-math">(z-a)^{-1}</span> 的系数”。计算题的第一步通常不是硬积分，而是找奇点、判极点阶数、算留数。

### 常用例子

1. <span class="kalax-inline-math">f(z)=1/z</span> 在 <span class="kalax-inline-math">0</span> 处有简单极点，留数为 <span class="kalax-inline-math">1</span>：

   <div class="kalax-display-math">\int_{|z|=1}\frac1z\,dz=2\pi i.</div>

2. 若

   <div class="kalax-display-math">f(z)=\frac{\sin z}{z^2},</div>

   因为

   <div class="kalax-display-math">\sin z=z-\frac{z^3}{3!}+\cdots,</div>

   所以

   <div class="kalax-display-math">\frac{\sin z}{z^2}=\frac1z-\frac z{3!}+\cdots,</div>

   从而

   <div class="kalax-display-math">\operatorname{Res}(f,0)=1.</div>

3. 若

   <div class="kalax-display-math">f(z)=\frac{e^z}{(z-a)^m},</div>

   则留数是 <span class="kalax-inline-math">e^z</span> 在 <span class="kalax-inline-math">a</span> 处 Taylor 展开中 <span class="kalax-inline-math">(z-a)^{m-1}</span> 的系数，即

   <div class="kalax-display-math">\operatorname{Res}(f,a)=\frac{e^a}{(m-1)!}.</div>

## 5. 开映射、最大模原理与共形映射

### Rouché 定理

**定理。** 设 <span class="kalax-inline-math">f,g</span> 在简单闭曲线 <span class="kalax-inline-math">C</span> 及其内部全纯。若在 <span class="kalax-inline-math">C</span> 上

<div class="kalax-display-math">|g(z)|&lt;|f(z)|,</div>

则 <span class="kalax-inline-math">f</span> 与 <span class="kalax-inline-math">f+g</span> 在 <span class="kalax-inline-math">C</span> 内部有相同个数的零点，按重数计算。

> [!tip] 怎么用
> 当一个函数是“主项 + 小扰动”时，Rouché 定理告诉我们零点个数不变。开映射定理的证明里，<span class="kalax-inline-math">f(z)-w</span> 可以看成 <span class="kalax-inline-math">f(z)-w_0</span> 加上小常数扰动。
### 开映射定理

**定理。** 若 <span class="kalax-inline-math">f</span> 在连通区域 <span class="kalax-inline-math">U</span> 上全纯且非常数，则 <span class="kalax-inline-math">f</span> 是开映射，即把开集映成开集。

**证明思路。** 取 <span class="kalax-inline-math">z_0\in U</span>，设 <span class="kalax-inline-math">w_0=f(z_0)</span>。若

<div class="kalax-display-math">f(z)-w_0=(z-z_0)^m g(z),\qquad g(z_0)\ne0,</div>

则在 <span class="kalax-inline-math">z_0</span> 附近，<span class="kalax-inline-math">f</span> 像 <span class="kalax-inline-math">z\mapsto z^m</span> 一样，把小圆盘映成包含 <span class="kalax-inline-math">w_0</span> 的邻域。严格证明可用 Rouché 定理：当 <span class="kalax-inline-math">w</span> 足够接近 <span class="kalax-inline-math">w_0</span> 时，<span class="kalax-inline-math">f(z)-w</span> 与 <span class="kalax-inline-math">f(z)-w_0</span> 在小圆周上有相同零点数，所以 <span class="kalax-inline-math">w</span> 落在 <span class="kalax-inline-math">f</span> 的像中。

### 最大模原理

**定理。** 若 <span class="kalax-inline-math">f</span> 在连通区域 <span class="kalax-inline-math">U</span> 上全纯且非常数，则 <span class="kalax-inline-math">|f|</span> 不能在 <span class="kalax-inline-math">U</span> 内部取得局部最大值。

**证明。** 若 <span class="kalax-inline-math">|f|</span> 在 <span class="kalax-inline-math">z_0</span> 取局部最大值，因 <span class="kalax-inline-math">f</span> 非常数，开映射定理说明 <span class="kalax-inline-math">f</span> 把 <span class="kalax-inline-math">z_0</span> 附近映成 <span class="kalax-inline-math">f(z_0)</span> 附近的开集。于是能找到值 <span class="kalax-inline-math">w</span> 满足 <span class="kalax-inline-math">|w|&gt;|f(z_0)|</span>，矛盾。

> [!note] 初学者提示
> 实变函数可以在内部有峰值，非零全纯函数不行。全纯函数的“大小”受边界控制。

### Schwarz 引理

设 <span class="kalax-inline-math">f:\mathbb D\to\mathbb D</span> 全纯，且 <span class="kalax-inline-math">f(0)=0</span>。则

<div class="kalax-display-math">|f(z)|\le |z|,\qquad |f'(0)|\le1.</div>

若某个非零点取等号，或 <span class="kalax-inline-math">|f'(0)|=1</span>，则

<div class="kalax-display-math">f(z)=e^{i\theta}z.</div>

**证明。** 令

<div class="kalax-display-math">g(z)=\begin{cases}
\dfrac{f(z)}z,&amp;z\ne0,\\
f'(0),&amp;z=0.
\end{cases}</div>

则 <span class="kalax-inline-math">g</span> 在 <span class="kalax-inline-math">\mathbb D</span> 全纯。固定 <span class="kalax-inline-math">0&lt;r&lt;1</span>，在圆周 <span class="kalax-inline-math">|\zeta|=r</span> 上有

<div class="kalax-display-math">|g(\zeta)|=\frac{|f(\zeta)|}{|\zeta|}&lt;\frac1r.</div>

由最大模原理，在 <span class="kalax-inline-math">|z|\le r</span> 内也有 <span class="kalax-inline-math">|g(z)|\le 1/r</span>。对固定的 <span class="kalax-inline-math">z\in\mathbb D</span>，令 <span class="kalax-inline-math">r\to1^-</span>，得到 <span class="kalax-inline-math">|g(z)|\le1</span>。因此

<div class="kalax-display-math">|f(z)|=|z|\,|g(z)|\le |z|.</div>

取 <span class="kalax-inline-math">z=0</span> 处的值，得到 <span class="kalax-inline-math">|f'(0)|=|g(0)|\le1</span>。若等号在内部出现，则 <span class="kalax-inline-math">g</span> 由最大模原理为常数，所以 <span class="kalax-inline-math">f(z)=e^{i\theta}z</span>。

### Mobius 变换

Mobius 变换是

<div class="kalax-display-math">T(z)=\frac{az+b}{cz+d},\qquad ad-bc\ne0.</div>

它们把 Riemann 球面

<div class="kalax-display-math">\widehat{\mathbb C}=\mathbb C\cup\{\infty\}</div>

双全纯地映到自身。

三个互异点可以唯一确定一个 Mobius 变换。常用工具是交比：

<div class="kalax-display-math">[z,z_1,z_2,z_3]
=\frac{(z-z_1)(z_2-z_3)}{(z-z_3)(z_2-z_1)}.</div>

它把 <span class="kalax-inline-math">z_1,z_2,z_3</span> 分别送到 <span class="kalax-inline-math">0,1,\infty</span>。

### Riemann 映射定理

**定理。** 若 <span class="kalax-inline-math">U\subsetneq\mathbb C</span> 是非空、单连通区域，则 <span class="kalax-inline-math">U</span> 与单位圆盘 <span class="kalax-inline-math">\mathbb D</span> 共形等价。更具体地，给定 <span class="kalax-inline-math">z_0\in U</span>，存在唯一全纯双射

<div class="kalax-display-math">f:U\to\mathbb D</div>

满足

<div class="kalax-display-math">f(z_0)=0,\qquad f'(z_0)&gt;0.</div>

> [!warning] 易错点
> Riemann 映射定理不适用于整个 <span class="kalax-inline-math">\mathbb C</span>。所以条件是 <span class="kalax-inline-math">U\subsetneq\mathbb C</span>，即真子区域。

证明通常依赖正规族、Montel 定理、Schwarz 引理等工具。笔记中相关证明可理解为如下逻辑链：

1. 构造一族从 <span class="kalax-inline-math">U</span> 到 <span class="kalax-inline-math">\mathbb D</span> 的单叶全纯函数。
2. 用 Cauchy 估计得到等度连续与局部有界。
3. 用 Arzela-Ascoli 或 Montel 定理取极限函数。
4. 证明极限函数仍然单叶且达到极值。
5. 用 Schwarz 引理证明归一化后的唯一性。

## 6. 多值函数、分支点与单值分支

### 为什么会有多值函数

典型多值函数包括

<div class="kalax-display-math">\log z,\qquad z^a=e^{a\log z},\qquad \sqrt{p(z)}.</div>

以 <span class="kalax-inline-math">\log z</span> 为例，若

<div class="kalax-display-math">z=re^{i\theta},</div>

则

<div class="kalax-display-math">\log z=\log r+i(\theta+2\pi k),\qquad k\in\mathbb Z.</div>

所以绕原点一圈后，函数值增加 <span class="kalax-inline-math">2\pi i</span>。

### 分支点与分支割线

直观地说，点 <span class="kalax-inline-math">a</span> 是多值函数 <span class="kalax-inline-math">f</span> 的分支点，若沿着围绕 <span class="kalax-inline-math">a</span> 的小圆做解析延拓，函数值不能回到原来的分支。

例如：

- <span class="kalax-inline-math">\sqrt z</span> 在 <span class="kalax-inline-math">0</span> 与 <span class="kalax-inline-math">\infty</span> 是分支点；绕 <span class="kalax-inline-math">0</span> 一圈，<span class="kalax-inline-math">\sqrt z</span> 变号，绕两圈才回来。
- <span class="kalax-inline-math">\log z</span> 在 <span class="kalax-inline-math">0</span> 与 <span class="kalax-inline-math">\infty</span> 是对数分支点；绕一圈后值增加 <span class="kalax-inline-math">2\pi i</span>，永远不会在有限圈后回到原值。
- <span class="kalax-inline-math">1/z</span> 在 <span class="kalax-inline-math">0</span> 是极点，不是分支点。

> [!warning] 易错点
> “奇点”不等于“分支点”。极点和本性奇点是单值全纯函数的局部问题；分支点是多值函数绕圈延拓后的全局问题。

为了得到单值分支，常常人为切开平面。例如 <span class="kalax-inline-math">\log z</span> 的主值分支通常定义在

<div class="kalax-display-math">\mathbb C\setminus(-\infty,0]</div>

上，并取

<div class="kalax-display-math">-\pi&lt;\arg z&lt;\pi.</div>

于是

<div class="kalax-display-math">\operatorname{Log}z=\log|z|+i\arg z.</div>

### <span class="kalax-inline-math">z^a</span> 的分支行为

定义

<div class="kalax-display-math">z^a=e^{a\log z}.</div>

若 <span class="kalax-inline-math">a=m/n\in\mathbb Q</span>，且 <span class="kalax-inline-math">m,n</span> 互素，则绕 <span class="kalax-inline-math">0</span> 一圈时

<div class="kalax-display-math">z^a\mapsto e^{2\pi i m/n}z^a.</div>

因此绕 <span class="kalax-inline-math">n</span> 圈回到原值，所以 <span class="kalax-inline-math">0</span> 是有限阶分支点。

若 <span class="kalax-inline-math">a\notin\mathbb Q</span>，则不存在有限圈数使 <span class="kalax-inline-math">e^{2\pi i a k}=1</span>，所以 <span class="kalax-inline-math">0</span> 是无限阶分支点，常称 logarithmic branch point。

若 <span class="kalax-inline-math">a\in\mathbb C\setminus\mathbb R</span>，绕圈还会改变模长，因为

<div class="kalax-display-math">e^{a(\log r+i\theta)}</div>

中的实部随 <span class="kalax-inline-math">\theta</span> 改变。

### 反三角函数的分支点

常用公式为

<div class="kalax-display-math">\arccos z=-i\log\left(z+i\sqrt{1-z^2}\right),</div>

<div class="kalax-display-math">\arcsin z=-i\log\left(iz+\sqrt{1-z^2}\right).</div>

因此它们的分支点来自 <span class="kalax-inline-math">\sqrt{1-z^2}</span>，即

<div class="kalax-display-math">z=1,\qquad z=-1,</div>

并且还要考虑 <span class="kalax-inline-math">\infty</span> 处的行为。

## 7. 解析延拓、单值化与单值性定理

### 解析延拓

设 <span class="kalax-inline-math">f</span> 是定义在小圆盘 <span class="kalax-inline-math">D</span> 上的全纯函数。若沿路径 <span class="kalax-inline-math">\gamma</span> 能在一串相交圆盘上连续地给出全纯函数，并且在重叠部分相等，就称 <span class="kalax-inline-math">f</span> 可沿 <span class="kalax-inline-math">\gamma</span> 解析延拓。

解析延拓的核心事实是：

**恒等定理。** 若两个全纯函数在连通区域内有聚点的一组点上相等，则它们处处相等。

因此解析延拓一旦存在，在每一步都是唯一的。

### Monodromy theorem

**单值性定理。** 若 <span class="kalax-inline-math">U</span> 单连通，函数芽可以沿 <span class="kalax-inline-math">U</span> 中任意路径解析延拓，并且延拓过程中没有遇到奇点，则延拓结果与路径无关，得到 <span class="kalax-inline-math">U</span> 上的单值全纯函数。

> [!note] 这在讲什么
> 单连通区域没有洞，因此绕来绕去的路径都能缩成一点。没有洞，就不会因为绕分支点而切换分支。

## 8. Riemann 曲面

### 曲面与复结构

一个拓扑空间 <span class="kalax-inline-math">S</span> 是曲面，若每个点都有邻域同胚于 <span class="kalax-inline-math">\mathbb C</span> 的开集。为避免病态，一般还要求 <span class="kalax-inline-math">S</span> Hausdorff 且第二可数。

一个 chart 是一对

<div class="kalax-display-math">(U,\phi),\qquad \phi:U\to\phi(U)\subset\mathbb C,</div>

其中 <span class="kalax-inline-math">U\subset S</span> 开，<span class="kalax-inline-math">\phi</span> 是同胚。

一个 atlas 是一族 chart，覆盖整个 <span class="kalax-inline-math">S</span>。若两个 chart 的坐标转换函数

<div class="kalax-display-math">\phi_j\circ\phi_i^{-1}:
\phi_i(U_i\cap U_j)\to\phi_j(U_i\cap U_j)</div>

全纯，则称 atlas 是解析 atlas。

一个 Riemann 曲面就是带有解析 atlas 的连通 Hausdorff 曲面。

> [!tip] 初学者提示
> Riemann 曲面不是“嵌在三维空间中的曲面”这个意思，而是“局部看起来像复平面，并且坐标变换是全纯的空间”。

### Riemann 曲面上的全纯函数

设 <span class="kalax-inline-math">S</span> 是 Riemann 曲面，<span class="kalax-inline-math">f:S\to\mathbb C</span>。若对每个 chart <span class="kalax-inline-math">(U,\phi)</span>，函数

<div class="kalax-display-math">f\circ\phi^{-1}:\phi(U)\to\mathbb C</div>

全纯，则称 <span class="kalax-inline-math">f</span> 在 <span class="kalax-inline-math">S</span> 上全纯。

若 <span class="kalax-inline-math">F:S_1\to S_2</span> 是两个 Riemann 曲面之间的连续映射，则 <span class="kalax-inline-math">F</span> 全纯，若对任意 chart <span class="kalax-inline-math">(U,\phi)</span>、<span class="kalax-inline-math">(V,\psi)</span>，局部表达

<div class="kalax-display-math">\psi\circ F\circ\phi^{-1}</div>

是复平面开集之间的全纯函数。

### Riemann 球面

Riemann 球面记作

<div class="kalax-display-math">\widehat{\mathbb C}=\mathbb C\cup\{\infty\}.</div>

它由两个坐标图覆盖：

<div class="kalax-display-math">z: \widehat{\mathbb C}\setminus\{\infty\}\to\mathbb C,</div>

<div class="kalax-display-math">w=1/z: \widehat{\mathbb C}\setminus\{0\}\to\mathbb C.</div>

交集上的坐标转换 <span class="kalax-inline-math">w=1/z</span> 全纯，因此它是 Riemann 曲面。

### 覆盖与分支覆盖

一个映射

<div class="kalax-display-math">\pi:X\to Y</div>

是覆盖映射，若每个 <span class="kalax-inline-math">y\in Y</span> 有邻域 <span class="kalax-inline-math">V</span>，使得

<div class="kalax-display-math">\pi^{-1}(V)=\bigsqcup_\alpha U_\alpha,</div>

且每个限制

<div class="kalax-display-math">\pi|_{U_\alpha}:U_\alpha\to V</div>

都是同胚。

若允许在离散点集上出现局部模型

<div class="kalax-display-math">z\mapsto z^{m+1},</div>

则得到分支覆盖。此时对应点称为分支点，分支阶为 <span class="kalax-inline-math">m</span>。

### 多值函数的 Riemann 曲面

多值函数的 Riemann 曲面可以理解为“把不同分支放在不同 sheet 上”。

#### 例 1：<span class="kalax-inline-math">\log z</span>

<span class="kalax-inline-math">\log z</span> 在 <span class="kalax-inline-math">\mathbb C\setminus\{0\}</span> 上有无限多个分支：

<div class="kalax-display-math">\log r+i(\theta+2\pi k),\qquad k\in\mathbb Z.</div>

其 Riemann 曲面是 <span class="kalax-inline-math">\mathbb C\setminus\{0\}</span> 的无限层覆盖。绕 <span class="kalax-inline-math">0</span> 一圈，sheet 编号 <span class="kalax-inline-math">k</span> 变成 <span class="kalax-inline-math">k+1</span>。单值化后，投影

<div class="kalax-display-math">\pi:S\to\mathbb C\setminus\{0\}</div>

是覆盖映射，而 <span class="kalax-inline-math">\log</span> 在 <span class="kalax-inline-math">S</span> 上成为单值全纯函数。

#### 例 2：<span class="kalax-inline-math">z^{m/n}</span>

若 <span class="kalax-inline-math">m,n</span> 互素，则 <span class="kalax-inline-math">z^{m/n}</span> 有 <span class="kalax-inline-math">n</span> 个分支。其 Riemann 曲面是 <span class="kalax-inline-math">n</span> 层覆盖，并在 <span class="kalax-inline-math">0</span> 与 <span class="kalax-inline-math">\infty</span> 有分支。绕 <span class="kalax-inline-math">0</span> 一圈时，分支循环置换。

单值化以后，在新曲面上可以写

<div class="kalax-display-math">w^n=z^m.</div>

#### 例 3：<span class="kalax-inline-math">\sqrt{p(z)}</span>

设

<div class="kalax-display-math">p(z)=\prod_{j=1}^r(z-a_j)^{m_j}.</div>

则 <span class="kalax-inline-math">\sqrt{p(z)}</span> 的分支点发生在 <span class="kalax-inline-math">m_j</span> 为奇数的 <span class="kalax-inline-math">a_j</span> 处。还要检查 <span class="kalax-inline-math">\infty</span>：若 <span class="kalax-inline-math">\deg p</span> 为奇数，则 <span class="kalax-inline-math">\infty</span> 也是分支点；若为偶数，则不是。

若所有分支都是简单分支，且把有限分支点与 <span class="kalax-inline-math">\infty</span> 处可能的分支点一起计入，总数为 <span class="kalax-inline-math">B</span>，则二层分支覆盖的 genus 由 Riemann-Hurwitz 公式给出：

<div class="kalax-display-math">2g-2=2(2\cdot 0-2)+B,</div>

因此

<div class="kalax-display-math">g=\frac{B-2}{2}.</div>

特别地，若 <span class="kalax-inline-math">p</span> 有 <span class="kalax-inline-math">2g+2</span> 个互异零点，曲线

<div class="kalax-display-math">w^2=p(z)</div>

对应的紧 Riemann 曲面的 genus 就是 <span class="kalax-inline-math">g</span>。

例如

<div class="kalax-display-math">w^2=(z-a_1)(z-a_2)(z-a_3)(z-a_4)</div>

有 <span class="kalax-inline-math">B=4</span> 个分支点，因此 genus 为 <span class="kalax-inline-math">1</span>，即拓扑上是环面。

> [!warning] 易错点
> 笔记 OCR 中曾把若干平方根例子的分支点写乱。判断 <span class="kalax-inline-math">\sqrt{p(z)}</span> 的原则很简单：奇数重零点是分支点；偶数重零点不是；无穷远点看次数奇偶。

### Monodromy 群

设多值函数有 <span class="kalax-inline-math">n</span> 个分支。绕某个分支点一圈会把这 <span class="kalax-inline-math">n</span> 个分支重新排列，因此得到一个置换。所有这种置换生成的子群

<div class="kalax-display-math">G\le S_n</div>

称为 monodromy 群。

例如 <span class="kalax-inline-math">\sqrt z</span> 有两个分支，绕 <span class="kalax-inline-math">0</span> 一圈交换它们，所以 monodromy 群为

<div class="kalax-display-math">\mathbb Z/2\mathbb Z.</div>

<span class="kalax-inline-math">\log z</span> 有无限多个分支，绕 <span class="kalax-inline-math">0</span> 一圈使 <span class="kalax-inline-math">k\mapsto k+1</span>，monodromy 群为

<div class="kalax-display-math">\mathbb Z.</div>

## 9. 用函数芽构造 Riemann 曲面

### 函数芽

设 <span class="kalax-inline-math">\Sigma</span> 是 Riemann 曲面，<span class="kalax-inline-math">a\in\Sigma</span>。若两个在 <span class="kalax-inline-math">a</span> 附近定义的亚纯函数 <span class="kalax-inline-math">f,g</span> 在某个更小邻域内相等，则称它们在 <span class="kalax-inline-math">a</span> 处有同一个函数芽，记为

<div class="kalax-display-math">[f]_a.</div>

所有亚纯函数芽构成集合

<div class="kalax-display-math">M=\{[f]_a:a\in\Sigma\}.</div>

定义投影

<div class="kalax-display-math">\pi:M\to\Sigma,\qquad \pi([f]_a)=a.</div>

### 拓扑

若 <span class="kalax-inline-math">f</span> 在开集 <span class="kalax-inline-math">D\subset\Sigma</span> 上亚纯，则定义

<div class="kalax-display-math">D(f)=\{[f]_b:b\in D\}.</div>

这些集合构成 <span class="kalax-inline-math">M</span> 的拓扑基。

> [!note] 直观理解
> 一个点 <span class="kalax-inline-math">[f]_a</span> 不只是 base point <span class="kalax-inline-math">a</span>，还记住了“在 <span class="kalax-inline-math">a</span> 附近是哪一个局部分支”。因此不同分支会成为不同 sheet 上的点。

### 为什么它是 Riemann 曲面

局部地，投影

<div class="kalax-display-math">\pi:D(f)\to D</div>

是同胚。于是可以把 <span class="kalax-inline-math">D</span> 上的坐标拉回到 <span class="kalax-inline-math">D(f)</span> 上，作为 <span class="kalax-inline-math">M</span> 的坐标。坐标转换来自 base 曲面 <span class="kalax-inline-math">\Sigma</span> 上的坐标转换，因此全纯。

所以 <span class="kalax-inline-math">M</span> 的每个连通分支都是一个 Riemann 曲面，并且多值函数在该曲面上成为单值亚纯函数。

### 与解析延拓的关系

若从一个函数芽 <span class="kalax-inline-math">[f]_a</span> 出发，沿路径解析延拓，就在 <span class="kalax-inline-math">M</span> 中得到一条路径。两个函数芽属于同一个连通分支，当且仅当可以通过解析延拓互相到达。

因此：

- 每个连通分支对应一个“最大解析延拓”的单值化空间；
- 加上适当分支点后，可得到分支覆盖；
- branch point 的离散性与 Hausdorff 性需要额外证明，通常是 Riemann 曲面理论中的技术部分。

## 10. 学习路线与检查清单

### 复分析计算题流程

1. 先找区域：函数在哪些点全纯？哪些点是奇点？
2. 判断区域拓扑：是否单连通？路径是否绕过奇点？
3. 若是积分题：找闭曲线内部的孤立奇点。
4. 对每个奇点算留数。
5. 用留数定理或 Cauchy 公式。

### 多值函数题流程

1. 写成 <span class="kalax-inline-math">e^{a\log z}</span> 或 <span class="kalax-inline-math">\sqrt{p(z)}</span> 的形式。
2. 找可能的分支点：对数的零点/极点，平方根的奇重零点，及 <span class="kalax-inline-math">\infty</span>。
3. 选 branch cuts，使剩余区域单连通。
4. 在该区域选定一个初值，得到单值分支。
5. 若要 Riemann 曲面，则说明有几层 sheet，绕分支点的 monodromy 是什么。

### 常见错误速查

- <span class="kalax-inline-math">1/z</span>：<span class="kalax-inline-math">0</span> 是极点，不是分支点。
- <span class="kalax-inline-math">\sqrt z</span>：<span class="kalax-inline-math">0,\infty</span> 是二阶换 sheet 的分支点，绕两圈回到原值。
- <span class="kalax-inline-math">\log z</span>：<span class="kalax-inline-math">0,\infty</span> 是对数分支点，绕有限圈不会回到原值。
- <span class="kalax-inline-math">z^{m/n}</span>：若 <span class="kalax-inline-math">m,n</span> 互素，有 <span class="kalax-inline-math">n</span> 个分支。
- <span class="kalax-inline-math">\sqrt{p(z)}</span>：只看 <span class="kalax-inline-math">p</span> 的奇数重零点；<span class="kalax-inline-math">\deg p</span> 为奇数时 <span class="kalax-inline-math">\infty</span> 也是分支点。
- Cauchy 公式中积分变量通常写 <span class="kalax-inline-math">z</span>，被求点写 <span class="kalax-inline-math">a</span> 或 <span class="kalax-inline-math">w</span>，不要混用。
- Cauchy 列要比较 <span class="kalax-inline-math">z_n</span> 与 <span class="kalax-inline-math">z_m</span>，不是比较 <span class="kalax-inline-math">z_n</span> 与某个极限。

> [!tip] 最后建议
> 这份笔记的核心不在“背公式”，而在三句话：全纯函数由局部幂级数控制；闭路积分由奇点控制；多值函数由绕分支点的 monodromy 控制。
